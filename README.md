# TennisTrain 🎾

一个专为移动端设计的网球训练网页应用，旨在通过交互式小游戏提升选手的节奏感和弹道预判能力。

## 核心功能
*   **节奏专家 (Rhythm Expert)**: 训练点击节奏与反应时间。
*   **弹道预测 (Trajectory Prediction)**: 训练对球落地后反弹轨迹的空间预判。
*   **移动端优先**: 针对手机端设计的 UI，支持触控反馈与全屏体验。
*   **Cloudflare 托管**: 极速部署，全球访问。

## 技术架构
*   **Frontend**: React + Vite + Tailwind CSS + Framer Motion
*   **Backend (Optional)**: Cloudflare Workers Python Runtime
*   **Deployment**: Cloudflare Pages

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
